# SecondHandMarket 🛍

중고 거래 플랫폼 클론 프로젝트 (KREAM 스타일)  
**Java + Spring Boot + MySQL + HTML/CSS/JS**

## 📌 주요 기능
- 회원가입 / 로그인
- 상품 조회 / 상세페이지
- 사이즈 옵션 및 입찰
- 거래 체결
- 마이페이지 (내 입찰 현황 등)

## 🧰 사용 기술
- **Backend**: Java 17, Spring Boot 3.x, Spring Data JPA, MySQL
- **Frontend**: HTML, CSS, JS (또는 Thymeleaf)
- **협업**: Git, GitHub, Git Flow 전략
